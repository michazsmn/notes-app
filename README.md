# Nextjs app with postgres and prisma

## Notes app synchronized with postgresql database

### Install dependencies
```bash
npm install
```

### Setup postgres
```bash
docker compose up
```

### Run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

### View app
```
http://localhost:3000
```


### Prisma studio
```bash
npx prisma studio
```

### Prisma create db
```bash
npx prisma db push
```
